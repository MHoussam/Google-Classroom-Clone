Expense Tracker

Christopher Yammine • Jul 18 (Edited Jul 18)
100 points
Due Jul 19, 5:00 PM
Your task is to implement a basic expense tracker using jQuery. An expense tracker is a tool that helps individuals or businesses keep track of their financial transactions. The expense tracker should have the following functionalities:
  • Display a form with input fields for the expense name, amount.
  • Include a button to add a new expense.
  • Validate the input fields to ensure they are not empty when the user clicks the "Add" button
  • .Add the expense to the expense list if the input fields are valid.
  • Display the list of expenses in a table format, showing the expense name and amount.
  • Show the total sum of all expenses at the bottom of the table.
  • Implement a delete button next to each expense to allow users to remove expenses from the list.
  • Apply CSS to style the expense tracker and make it visually appealing.
You are required to use jQuery for the JavaScript part of the implementation. Submit your assignment by providing the respective repo link.
